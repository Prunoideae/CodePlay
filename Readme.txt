需要pip install的包已经在源码中写明

要在命令行里使用CodePlay播放歌单，你需要
python [CodePlay文件路径] [歌单文件路径]
你可以使用ftype和assoc来关联歌单文件的扩展名，歌单文件的扩展名和歌单是否可以被加载无关，只要内部是有效的代码即可播放。
要查看歌单的错误信息，还是要用命令行运行的
基本的歌单示例已经在Examples里有写明了

默认的快捷键是(Shift+Ctrl+)
R(resume)恢复播放 P(pause)暂停播放 X(next)下一首 B(before)上一首 ↓调低音量 ↑调高音量
你可以修改歌单来自定义快捷键和各种情况的行为。
